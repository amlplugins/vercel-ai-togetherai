# @amlplugins/vercel-ai-togetherai

Native `@ai-sdk/togetherai` SDK re-exported under the `@amlplugins` namespace.

> Vercel AI SDK Together AI provider — 200+ open-source LLMs and embedding models via Together inference.

## Install

```bash
npm install @amlplugins/vercel-ai-togetherai
```

## Use

```ts
import { /* named exports from @ai-sdk/togetherai */ } from "@amlplugins/vercel-ai-togetherai";
// or
import sdk from "@amlplugins/vercel-ai-togetherai";
```

This package additionally re-exports `ai` under the `secondary` namespace.

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
